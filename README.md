### Installation

* Download and extract the [latest version of Movies catalog](https://github.com/paratagas/movies-catalog)
* Install the dependencies and devDependencies:
```sh
$ cd movies-catalog
$ npm install
```

---

### Launching
```sh
$ npm start
```

After that your web application is available at:

http://localhost:3000
